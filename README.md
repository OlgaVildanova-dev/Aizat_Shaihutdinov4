# Random Password Generator

**Автор:** Айзат Шайхутдинов
**Язык:** Python 3.x

## Описание

Простое GUI-приложение для генерации случайных паролей с настройкой длины и типов символов. История паролей сохраняется в файл `history.json`.

## Требования

- Python 3.x
- Библиотека Tkinter (встроена в Python)
- random, json (встроены)

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/yourname/password-generator.git
   cd password-generator
   ```
2. Запустите приложение:
   ```bash
   python main.py
   ```
3. Используйте ползунок для выбора длины, отметьте нужные типы символов и нажмите «Сгенерировать пароль».
4. Для сохранения пароля в историю нажмите «Добавить в историю».
5. История сохраняется в `history.json`.
